This app is developed to aid players, via Slack, in the game Dungeon World--a fantasy table-top role-playing games based on Dungeons and Dragons. This app takes in a request from a Slack channel via various commands and returns useful gameplay information and functions. Such commands include rolling various sided-dice; creating characters with stats, weapons, and armor; and attaching these characters and dice rolls to particular Slack users. Users can have multiple characters which they can switch between at any time--for example, if the user has multiple games with different characters for each.

ReRoot is an App geared towards allowing users to organize research in a tree-like structure. This allows users to better visualize their research progress and locate saved articles quickly. ReRoot is designed both for individual users and users who wish to work in teams on larger research projects. This project includes strict permissions dictating who can see what content; rerouting the user if he or she is unauthorized.

A large part of this project involved dealing with the speed issues associated with generating large amounts of data. Caching and eager loading proved effective but was incompletely implemented due to the recursive nature of the data. Nevertheless, most of the speed issues resulted from the large amount of nested partials and was resolved by converting some of these partials into React components.
